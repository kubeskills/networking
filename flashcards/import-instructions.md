# Flashcard Import Instructions

## For Anki

1. Download and install [Anki](https://apps.ankiweb.net/)
2. Open Anki and create a new deck named "Kubernetes Networking"
3. File → Import → Select `deck.txt`
4. Set import format to:
   - Type: "Basic"
   - Fields separated by: New line
   - Allow HTML in fields: No
5. Click Import
6. Start reviewing!

## For Physical Flashcards

Print `deck.txt` and cut into cards:
- Front: Question (odd lines)
- Back: Answer (even lines)

## For Quizlet

1. Go to [Quizlet.com](https://quizlet.com)
2. Create a new study set
3. Copy-paste from `deck.txt` (question/answer pairs)
4. Set delimiter to "new line"

## Study Schedule (Spaced Repetition)

- **Day 0 (Today):** Review all cards once
- **Day 2:** Review again (20% retention boost)
- **Week 1:** Review again (50% retention boost)  
- **Month 1:** Review again (80% long-term retention)

## Adding Custom Cards

Format:
```
Question here?
Answer here

Next question?
Next answer
```

Keep questions concise and answers actionable.
